# Weather App by Gopinath

## Requirements
 - php 8.2 or above
 - composer
 - codeigniter4 4.7 or above

## Setup Instructions
 - run "git clone https://github.com/gopi-s-2k/weather-app-ci"
 - inside the project directory run "composer install" to install vendor packages
 - serve the app using command "php spark serve"

## Serving port
 - By default the app will be served in 8080
 - If changing the port in "php spark serve" command, make it consistant with .env file's app_baseURL
 
## Open Weather API integration
    I have included .env file in github repository which contains my api key.
    So, No specific configuration is needed for weather api integration.
